You don't need install this program. Only double click to exe file. 

1. To printscreen all screen click button "All"
2. To printscreen part of screen click "Part" next click and hold left mouse button in first point of screen you want then drag 
and release left mouse button. 

Screenshot save on desktop in jpg. 
Screenshoter work on main screen (if you've got more then one screen). 

Work with Win 7 and Win 8 (I don't test it with other Windows).
